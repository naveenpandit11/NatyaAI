# 🖐 AI-Based Bharatiya Natya Mudra Recognition System

An AI-powered web application that detects and recognizes **Bharatiya Natya Mudras** in real-time using Computer Vision and Machine Learning.

This project combines **MediaPipe hand tracking**, **Machine Learning**, and a **modern Tailwind web interface** to create an interactive gesture recognition system.

---

## 🚀 Features

✅ Real-time hand gesture detection  
✅ AI-based Mudra classification  
✅ Confidence score prediction  
✅ Responsive Tailwind UI  
✅ Flask backend API  
✅ Camera-based live recognition  
✅ Lightweight ML model (no GPU required)
